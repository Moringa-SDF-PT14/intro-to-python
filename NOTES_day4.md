## PyTest

Things to remember when working with tests in pytest:

1. All test files must be prefixed `(test_xyz.py)`  or suffixed `(xyz_test.py)` with the `test` keyword.
2. No need for wrappers (`describe`) or (`test`). Simple function prefixed with `test_`.

## Object Oriented Programming (OOP)

A programming paradigm that focuses on treating programs as a function of their data.

### Aspects / Components of OOP

- Classes (Blueprints upon which objects can be created). To define and group behavior that all object implementations will follow.

    1. `constructor` - default function called when a new object is being created.
    2. `properties` - variables that hold values for objects [instance, class]
    3. `methods` - functions in the class [instance - apply to object, class]

self -> representation of the current instance

- Object / Instances - Implementation of a class
